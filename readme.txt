== Installation ==
**WooCommerce Visual Page Customiser** requires the [WooCommerce](https://wordpress.org/plugins/woocommerce/ "WooCommerce") plugin (at least version 2.5.5) to be installed. Tested from WooCommerce 2.5.5 to 3.6.5.

= Via WordPress =
1. From the WordPress Dashboard, go to Plugins > Add New
2. Search for 'WooCommerce Visual Page Customiser' and click Install. Then click Activate.
3. TBW

= Manual =
1. Upload the folder /woo-better-customer-list/ to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. TBW
